HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git add -A: adds a change in the working directory to the staging area.
  • git commit -m: commit your staged content as a new commit snapshot.
  • git pull origin main: pull changes from the origin remote, master branch and merge them to the locally checked-out branch.
  • git push origin branch: a git command that pushes a local branch(es) to a remote repository (origin).
  • git status: checks what branch we are currently on.
  • git checkout -b (branch-name): creates a new branch and switches to it. You do not need to include "-b" if branch already exists.

JavaScript

  • Variable: A named container that allows us to store data in our code.
  • Array: A single variable that is used to store elements of different data types.
  • For Loop: An iterative statement which you use to check for certain conditions and then repeatedly execute a block of code as long as those conditions are met.
  • Function: A block of code designed to perform a particular task
  • Control flow: The order in which a computer executes code in a script.
